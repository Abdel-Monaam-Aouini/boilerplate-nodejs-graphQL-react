# boilerplate using NodeJS GraphQL ReactJS

#
clone the repository :

```sh
    git clone git@github.com:Monaam12/boilerplate-nodejs-graphQL-react.git
```
# server

Run :

```sh
    cd server && yarn
```

```sh
    docker-compose up --build -d mongodb
```

```sh
    yarn start
```

# client

Run :

```sh
    cd client && yarn
```

```sh
    yarn start
```
